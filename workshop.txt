description=Pry Door Advanced - Keybind Only (v3.6.0)
description=
description=Description:
description=This mod allows players to forcibly open (pry) locked objects in Project Zomboid using a keybind. The main focus is on doors, garage doors, and vehicles. Windows are included in the code but are currently non-functional.
description=
description=Main Features:
description=1 Pry locked doors, including standard doors and IsoThumpable doors.
description=2 Pry garage doors with realistic animations and sounds.
description=3 Pry vehicle doors, detecting the nearest locked door within range.
description=4 Only locked objects can be pried; unlocked or already open objects are ignored.
description=5 Supports multiplayer, syncing door, garage, and vehicle states across players.
description=6 Compatible with various prying tools: Crowbar, Axe, Screwdriver, Hammer, and Wrench.
description=7 Grants XP for Strength, Woodwork, or Mechanics depending on the object.
description=8 Use the ` (backtick) key to start prying when near a valid target.
description=
description=Notes:
description=
description=Windows are still listed in the code but not functional. Focus is on doors, garages, and vehicles.
description=
description=A prying tool must be in the player’s inventory to perform the action.
description=
description=How to Use:
description=1 Approach a locked door, garage, or vehicle.
description=2 Make sure you have a prying tool in your inventory.
description=3 Press ` (backtick) to start prying.
description=4 Animations and sounds will play during the action.
